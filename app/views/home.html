This application has been designed during my Full Stack Web Development Specialization Capstone Project session The data are stored in a "noSQL" database and are accessd via a loopback server located in the IBM Bluemix cloud The url access of the REST API is : here

This application manipulates foolowing objects :

  • Chronogram : is the highest level entity, it is equivalent to a project
  • Task : is an elementary unit of work, a Chronogram is composed of many tasks
  • History : is a timestamped text element written by the person realizin the task in order to inform all the connected people

To begin, click on chronograms in the header menu
